Processo di Comunicazione

Socket TCP

La comunicazione tramite socket TCP avviene in questo modo:

  1. Creazione della socket lato server.
  2. Binding dell'indirizzo IP e della porta al server.
  3. Messa in ascolto del server e accettazione della connessione dal client.
  4. Scambio bidirezionale di dati tra server e client.
  5. Chiusura della connessione una volta terminata la comunicazione.

Socket UDP

La comunicazione tramite socket UDP avviene in questo modo:

  1. Creazione della socket senza connessione.
  2. Invio di dati dal client al server.
  3. Ricezione dei dati lato server (opzionalmente il server risponde al client).
  4. Chiusura della comunicazione senza la necessit√† di terminare formalmente la connessione.
